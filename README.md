# nfl-trends
Our website looks at data on NFL games from 2000-2025 in order to provide betting analysis, game predictions, and stadium information

### Features
#### Stadium Map
The Stadium Map tab has an interactive map that allows users to see the amount of overs/unders hit at a given stadium

#### Betting Analysis
Allows users to analyze different betting strategies based on certain factors including stadium type, weather, and game week

#### ROI Calculator
Allows users to simulate betting strategies and includes customizable betting amounts and year ranges

#### Game Predictor
Generates predictions of games using a percent likelihood based on customizable factors like wins/losses, offense/defense ranks, and game week

#### Cluster Analysis
Allows users to select two factors and provides a cluster chart, with optional use of principal component analysis

#### My Betting Library
Allows users to save their personal bets to the website
